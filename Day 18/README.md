## **CSS Function**

### **Linear Gradient**

-   It is a CSS function.
-   First parameter is a angle of the gradient.
-   Consists of two or more colors.

```css
footer {
    background: linear-gradient(70deg, rgb(24, 24, 24), rgb(25, 29, 29));
}
```
